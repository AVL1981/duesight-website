Notaris Due Diligence API

Automatiseer uw Wwft-verplichtingen met de DueSight Notaris API. UBO-identificatie, sanctiescreening en KvK-verificatie via een enkele REST API — direct integreerbaar in uw notarissoftware.

Base URL: https://api.duesight.nl/v1/notaris
Format: JSON · Auth: Bearer Token · Rate Limit: 100 req/min

🔐 Authenticatie

Alle API calls vereisen een Bearer token in de Authorization header. Tokens zijn beschikbaar via uw DueSight dashboard.

Request Header HTTP 
Authorization: Bearer ds_ntrs_live_sk_xxxxxxxxxxxx
Content-Type: application/json

⚡ Quickstart

Een volledige Wwft-check in 3 regels code:

cURL Voorbeeld BASH 
curl -X POST https://api.duesight.nl/v1/notaris/full-check \
  -H "Authorization: Bearer ds_ntrs_live_sk_xxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "kvk_number": "30204462",
    "transaction_type": "overdracht",
    "include_ubo": true,
    "include_sanctions": true
  }'

POST /v1/notaris/full-check

Volledige Wwft-check in één API call: combineert UBO, sanctiescreening, KvK-verificatie en adresvalidatie. Retourneert een compleet dossier met audit trail.

Request Body

POST Body JSON 
{
  "kvk_number": "30204462",
  "transaction_type": "overdracht",  // overdracht | oprichting | fusie
  "include_ubo": true,
  "include_sanctions": true,
  "include_address_verification": true,
  "additional_persons": [           // Optioneel: extra personen screenen
    { "name": "J. de Vries", "role": "koper" }
  ]
}
